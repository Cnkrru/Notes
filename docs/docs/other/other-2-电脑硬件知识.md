# 电脑硬件知识

## 一、电脑配件与系统基础

### 1. 电脑优化操作

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | Win+I → 储存 → 储存感知 | 天数改为"一天" |
| 2 | 应用 → 启动 | 关闭没用的启动项 |
| 3 | 隐私 → 常规 | 关闭常规四项 |
| 4 | Win+R → 输入"MRT" | 系统自带恶意软件查找器 |
| 5 | Windows 更新 → 高级选项 | 关闭"传递优化" |
| 6 | Win+R → 输入"services.msc" | 找到 Windows Update，关闭服务 |

## 二、CPU 核心概念

| 概念 | 比喻/说明 | 应用场景 |
|------|----------|---------|
| **核心** | 超级计算机/工厂 | 处理数据：硬盘→内存（原材料）→CPU 加工→GPU 组装→输出显示 |
| **单核性能** | 处理单个事务 | 玩游戏等单任务场景 |
| **多核性能** | 同时处理多个事项 | 视频剪辑、渲染等多任务场景 |
| **逻辑处理器** | 多只手帮核心干活 | 提高性能，如 6 核心 -12 逻辑处理器 |
| **数据宽度** | 车道数量 | 32 位（32 车道）vs 64 位（64 车道） |
| **32 位系统** | 最大处理 2³²，承载 4GB 内存 | 文件存放：x86 |
| **64 位系统** | 最大处理 2⁶⁴，支持 8/16GB+ 内存 | 文件存放：x64，可兼容 32 位 |
| **外频** | CPU 与主板传输基准频率 | 主流固定 100MHz，改变会造成不必要影响 |
| **主频** | 心脏跳动速率（主频=外频×倍频） | 主频高≠性能好 |

**CPU 与 GPU 分工**：
- CPU：主管计算
- GPU：管理图像

**主板**：相当于人的骨架和神经系统，连接各个部件并传递信息

## 三、CPU、主板、内存兼容性

| Intel CPU | 适配主板芯片组 | PCIe 类型 | 适配内存类型 |
|-----------|---------------|-----------|-------------|
| 80286 | 82340DX、82340SX | 无 PCIe，支持 ISA 总线 | 30pin SIMM |
| 80386 | 82350 EISA、82360SL | 无 PCIe，支持 EISA/ISA总线 | 30pin SIMM |
| 80486 | 420TX、420EX、420ZX | PCI 2.0/2.1 | FPM 内存 |
| Pentium 4/Celeron D | 945P、965P、X48 | PCIe 1.1（X48 支持 PCIe 2.0） | DDR2 |
| Core 2 系列 | G41、P35、X58 | PCIe 2.0 | G41/P35 支持 DDR2/DDR3，X58 支持 DDR3 |
| 第 6-7 代酷睿 | B150、Z170、B250、Z270 | PCIe 3.0 | DDR4、DDR3L |
| 第 8-9 代酷睿 | Z370、B360、Z390 | PCIe 3.0 | DDR4 |
| 第 10 代酷睿 | Z490、B460、H410、Z590 | PCIe 3.0（H410 支持 2.0） | DDR4 |
| 第 12-14 代酷睿 | Z690、B660、Z790、B760 | PCIe 3.0/4.0 | DDR4、DDR5 |
| Core Ultra 系列 | Z890、B860 | PCIe 4.0/5.0 | DDR5 |

## 四、硬件兼容性规则

| 类型 | 向下兼容 | 向上兼容 | 限制 |
|------|---------|---------|------|
| **CPU** | 同接口+BIOS+ 供电支持才兼容 | 几乎不可行 | 接口是核心，一般不能上下兼容 |
| **内存** | 不支持 DDR4/5 兼容 | 不可行 | 接口不同，完全不能混用 |
| **显卡** | PCIe 接口向下兼容 | 完全可以 | 显卡适配性极好，高端卡插老主板会有性能缺失 |

## 五、DDR4 vs DDR5 内存对比

| 维度 | DDR4 | DDR5 |
|------|------|------|
| 数据传输速度 | 2133-3200 MT/s | 4800-6400 MT/s |
| 理论带宽 | 最高 25.6 GB/s | 最高 51.2 GB/s |
| CAS 延迟 | 13-16 | 36-40 |
| 工作电压 | 1.2V | 1.1V |
| 单条最大容量 | 32GB | 128GB |
| 架构特点 | 单 64 位通道 | 双 32 位子通道 |

## 六、硬盘对比

| 硬盘类型 | 接口/协议 | 顺序读写速度 | 核心特点 | 适用场景 | 常见容量 |
|---------|----------|-------------|---------|---------|--------|
| SATA 机械硬盘 | SATA 3.0 | 读取 100-200MB/s<br>写入 80-150MB/s | 容量大、价格低，有机械部件，抗震性差，噪音明显 | 数据备份、监控存储、大容量影音归档 | 1TB-10TB |
| SATA 固态硬盘 | SATA 3.0 | 读取约 500-550MB/s<br>写入约 450-520MB/s | 无机械部件，抗震静音，受 SATA 接口限速 | 老电脑升级、基础办公主机 | 256GB-4TB |
| M.2 NVMe 固态硬盘 | PCIe Gen4/Gen5 + NVMe | Gen4：读取 3000-7300MB/s<br>Gen5：读取可达 14900MB/s | 体积小、速度超快，支持 HMB 等提速技术 | 新装机系统盘、游戏盘、专业视频剪辑 | 500GB-8TB |
