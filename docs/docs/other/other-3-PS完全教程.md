# PS 完全教程

> Photoshop 完全教程，从基础到高级

## 一、PS 基础操作

### 1. 界面介绍

- **菜单栏**：文件、编辑、图像、图层、选择、滤镜、视图、窗口、帮助
- **工具栏**：左侧的工具集合，包括选择工具、画笔工具、橡皮擦工具等
- **选项栏**：位于工具栏下方，显示当前工具的选项
- **面板**：右侧的各种面板，如图层、通道、路径、历史记录等
- **工作区**：中央的编辑区域

### 2. 常用工具

| 工具 | 快捷键 | 功能 |
|------|--------|------|
| 移动工具 | V | 移动图层或选区 |
| 矩形选框工具 | M | 创建矩形选区 |
| 椭圆选框工具 | M (Shift) | 创建椭圆选区 |
| 套索工具 | L | 手动创建选区 |
| 魔棒工具 | W | 根据颜色创建选区 |
| 快速选择工具 | W (Shift) | 快速选择相似区域 |
| 裁剪工具 | C | 裁剪图像 |
| 画笔工具 | B | 绘制线条 |
| 橡皮擦工具 | E | 擦除像素 |
| 渐变工具 | G | 创建渐变效果 |
| 文字工具 | T | 添加文字 |
| 吸管工具 | I | 吸取颜色 |
| 缩放工具 | Z | 放大或缩小视图 |
| 抓手工具 | H | 移动视图 |

### 3. 基本操作

- **新建文件**：文件 → 新建
- **打开文件**：文件 → 打开
- **保存文件**：文件 → 保存 / 另存为
- **导出文件**：文件 → 导出
- **复制图层**：右键图层 → 复制图层
- **删除图层**：选中图层 → 按 Delete 键
- **调整图层顺序**：拖动图层到合适位置
- **合并图层**：选中多个图层 → 右键 → 合并图层

## 二、图层操作

### 1. 图层类型

| 图层类型 | 功能 |
|---------|------|
| 普通图层 | 基本的图像图层 |
| 调整图层 | 调整图像的色彩、亮度等 |
| 填充图层 | 填充纯色、渐变或图案 |
| 形状图层 | 创建矢量形状 |
| 文字图层 | 包含文字内容 |
| 智能对象 | 保留原始图像数据，可无损编辑 |

### 2. 图层样式

- **投影**：添加阴影效果
- **内阴影**：添加内部阴影效果
- **外发光**：添加外部发光效果
- **内发光**：添加内部发光效果
- **斜面和浮雕**：添加立体效果
- **光泽**：添加光泽效果
- **颜色叠加**：叠加颜色
- **渐变叠加**：叠加渐变
- **图案叠加**：叠加图案
- **描边**：添加边框

### 3. 图层混合模式

| 混合模式 | 效果 |
|---------|------|
| 正常 | 默认模式 |
| 溶解 | 产生颗粒效果 |
| 变暗 | 只保留较暗的像素 |
| 正片叠底 | 相乘效果，使图像变暗 |
| 颜色加深 | 加深颜色 |
| 线性加深 | 线性加深效果 |
| 深色 | 比较颜色通道，保留较暗的 |
| 变亮 | 只保留较亮的像素 |
| 滤色 | 相反的正片叠底，使图像变亮 |
| 颜色减淡 | 减淡颜色 |
| 线性减淡 | 线性减淡效果 |
| 浅色 | 比较颜色通道，保留较亮的 |
| 叠加 | 结合正片叠底和滤色 |
| 柔光 | 柔和的叠加效果 |
| 强光 | 强烈的叠加效果 |
| 亮光 | 类似曝光效果 |
| 线性光 | 线性光效果 |
| 点光 | 点光效果 |
| 实色混合 | 产生强烈的色彩对比 |
| 差值 | 差值效果 |
| 排除 | 类似差值，但效果较柔和 |
| 色相 | 只保留色相 |
| 饱和度 | 只保留饱和度 |
| 颜色 | 保留色相和饱和度 |
| 明度 | 只保留明度 |

## 八、实用技巧

### 1. 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl + N | 新建文件 |
| Ctrl + O | 打开文件 |
| Ctrl + S | 保存文件 |
| Ctrl + Shift + S | 另存为 |
| Ctrl + Z | 撤销 |
| Ctrl + Shift + Z | 重做 |
| Ctrl + C | 复制 |
| Ctrl + V | 粘贴 |
| Ctrl + X | 剪切 |
| Ctrl + D | 取消选区 |
| Ctrl + A | 全选 |
| Ctrl + T | 自由变换 |
| Ctrl + E | 合并图层 |
| Ctrl + Alt + E | 盖印图层 |
| Ctrl + Shift + N | 新建图层 |
| Ctrl + Alt + Shift + N | 新建图层（无对话框） |

### 2. 图层蒙版

- **创建图层蒙版**：选中图层 → 点击图层面板底部的添加蒙版按钮
- **编辑图层蒙版**：用画笔工具在蒙版上绘制，黑色隐藏，白色显示，灰色半透明
- **应用图层蒙版**：右键蒙版 → 应用图层蒙版
- **删除图层蒙版**：右键蒙版 → 删除图层蒙版

### 3. 智能对象

- **创建智能对象**：选中图层 → 右键 → 转换为智能对象
- **编辑智能对象**：双击智能对象缩略图
- **栅格化智能对象**：右键智能对象 → 栅格化图层

### 4. 批量处理

- **动作**：窗口 → 动作，录制动作后批量应用
- **批处理**：文件 → 自动 → 批处理

### 5. 插件

- **Nik Collection**：专业的滤镜插件
- **Topaz Labs**：降噪和锐化插件
- **Portraiture**：人像磨皮插件
- **Knockout**：抠图插件

3. **图层排列**：调整各个元素的图层顺序
4. **融合处理**：使用图层蒙版和混合模式使元素融合
5. **光影调整**：调整各个元素的光影，使整体协调
6. **色彩统一**：使用色彩平衡和色相/饱和度统一整体色彩

## 十、常见问题及解决方案

| 问题 | 解决方案 |
|------|----------|
| 图像模糊 | 使用锐化工具增强清晰度 |
| 颜色偏差 | 使用色彩平衡和色相/饱和度调整 |
| 图层过多 | 合并相似图层，使用智能对象 |
| 内存不足 | 减少历史记录步数，关闭不必要的面板 |
| 保存文件过大 | 适当降低图像分辨率，选择合适的文件格式 |
| 抠图边缘不自然 | 使用羽化和调整边缘工具 |
| 文字模糊 | 确保文字图层分辨率足够，使用适当的字体 |
| 滤镜效果不理想 | 调整滤镜参数，使用图层蒙版控制效果范围 |
