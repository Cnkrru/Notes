# 正则表达式方法

## 1. test() 方法

| 方法 | 语法 | 说明 |
|------|------|------|
| test() | `regObj.test(被检测的字符串)` | 用来查看正则表达式与指定的字符串是否匹配，返回布尔值 |

---

## 2. replace() 方法

| 方法 | 语法 | 说明 |
|------|------|------|
| replace() | `字符串.replace(正则表达式, 替换的内容)` | 用来替换字符串中匹配正则表达式的部分，返回替换后的新字符串 |

### 高级用法

| 功能 | 语法 | 说明 |
|------|------|------|
| 全局替换 | `str.replace(/andy/g, 'baby')` | 使用 `g` 标志进行全局替换 |
| 忽略大小写 | `str.replace(/andy/i, 'baby')` | 使用 `i` 标志忽略大小写 |
| 全局忽略大小写 | `str.replace(/andy/gi, 'baby')` | 同时使用 `g` 和 `i` 标志 |
| 替换所有数字 | `str.replace(/\d/g, '*')` | 使用 `\d` 匹配所有数字并替换为 `*` |
