## 工作模式
1. SCON寄存器
    - SCON.5 -> SM2
    - SCON.6 -> SM1
    - SCON.7 -> SM0
    - 0/1控制串口工作模式,2用于多机通信，控制是否收发信息
2. 工作模式
    - 00：同步移位寄存器（用于拓展IO口）
    - 01：8位异步收发（波特率可变，由定时器控制）
    - 10：9位异步收发（波特率为f/64或f/32）
    - 11：9位异步收发（波特率可变，由定时器控制）

    - SM=1,只有第9位=1时，接收数据
    - SM=0,无脑接收数据
3. RT
    - SCON.0 -> RI
    - SCON.1 -> TI
    - 接收完数据由硬件置1
4. RB/TB
    - SCON.2 -> RB8
    - SCON.3 -> TB8
    1. 数据位
        - 0:串口8位数据位
        - 1:串口9位数据位
    2. 奇偶校验位 哦
5. REN
    - SCON.4 -> REN
    - 0:禁止接收
    - 1:允许接收
6. SMOD
    - PCON.7 -> SMOD
    - 波特率选择位
    - SMOD=1相比SMOD=0，波特率翻倍
7. 工作模式详解
    1. 方式0：8位同步
        - 用于外拓IO
        - 无起始位和终止位
        - 波特率：f/12
        - 收放完数据后RI/TI置1
    2. 方式1：10位双机通信
        - 用于双机通信
        - 1位起始位和1位终止位
        - 波特率：2^SMOD/32*定时器T1溢出率
    3. 方式2：11位异步通信
        - 用于异步双机通信
        - 1位起始位0和8位数据位，1位可控1/0的第9位数据，1位终止位
        - 波特率：2^SMOD/64*f
    3. 方式3：9位异步通信
        - 波特率：2^SMOD/32*定时器T1溢出率
        - 其余与方式2一样
    >溢出率：（f/12）/（256-x）
8. 波特率制定方法
    - 收放双方波特率必须一致
9. 程序写法
    - 甲机发送程序（外部中断式）
```asm
ORG 0000H
SJMP MAIN
ORG 0003H
SJMP INT0
ORG 000BH
SJMP TIM0

MAIN:
    MOV TOMD,#20H
    MOV TL1,#0F4H
    MOV TH1,#0F4H
    MOV PCON,#00H
    MOV SCON,#40H

    SETB EA
    SETB ES
    SETB TR1

    MOV SUBF,20H
    SJMP $
COMINT:
    CLR TI
    CLR EA
    CLR ES
    RETI 
    END

```    
    - 甲机发送程序（轮询）
```asm
ORG 0000H
MAIN:
    MOV TOMD,#20H
    MOV TL1,#0F4H
    MOV TH1,#0F4H
    MOV PCON,#00H
    MOV SCON,#40H

    SETB TR1
    MOV SBUF,20H
    JNB TI,$
    CLR T1
    END
```
    - 已机接收程序
```asm
0RG  0000H
SJMP MAIN
ORG  0023H
SJMP COMINT
ORG  0030H

MAIN: 
    MOV TMOD,#20H
    MOV TL1,#0F4H
    MOV TH1,#0F4H
    MOV PCON,#00H
    MOV SCON,#50H
    SETB ES 
    SETB EA
    SETB TR1 
    SJMP  $
COMINT:
    CLR RI
    MOV 2OH,SBUF
    CLR EA
    CLR ES
    RETI 
    END
```
