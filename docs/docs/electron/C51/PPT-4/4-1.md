## 查表(0-9平方数)

;查表，0-9平方数,下标索引变量x在寄存器A,用数据指针DPTR查找
;步骤：
;1. 制表
;2. 给数据指针数组的首地址
;3. @A+DPTR查找
```asm
ORG 0000H
;程序
MAIN:
    MOV DPTR #TAB           ;给初地址
    MOVC A,@A+DPTR          ;给指针
    MOV 30H,A               ;给结果

ORG 1000H
TAB:
    DB 00H,01H,04H,09H,10H
    DB 19H,24H,31H,40H,51H
```
