## 中断
>TCON寄存器
1. TIM
    - TF0/1:TIM溢出标志位
        - 0：TCON.5 -> 8DH
        - 1：TCON.7 -> 8FH
2. INFO
    - IE0/1:外部中断请求标志位
        - 0:TCON.1 -> 89H
        - 1:TCON.3 -> 8BH
    - IT0/1:外部中断触发方式
        - 0:TCON.0 -> 88H   电平触发
        - 1:TCON.2 -> 8AH   跳沿触发
>SCON寄存器
3. USAT
    - TI/RI:串口发送/接收完成标志位
        - RI:SCON.0 -> 98H
        - TI:SCON.1 -> 99H

## 中断允许寄存器(enable<>)
>高电平有效
1. 中断总开关
    - EA
2. 定时器溢出中断允许位
    - ET0
    - ET1
    - ET2
3. 外部中断允许位
    - EX0
    - EX1
4. 串口中断允许位
    - ES

操作：
>CLR <E寄存器>
>SETB <E寄存器>

## 中断优先级寄存器
>高电平为高优先级
1. 定时器中断优先级控制器
    - PT0
    - PT1
    - PT2
2. 外部中断优先级控制器
    - PX0
    - PX1
3. 串口中断优先级控制器
    - PS

操作：
>CLR <P寄存器>
>SETB <P寄存器>

## 寄存器地址

    |中断源|入口地址|
    |---|---|
    |INT0|0003H|
    |TIM0|000BH|
    |INT1|0013H|
    |TIM1|001BH|
    |USAT|0026H|
    |TIM2|002BH|

>相邻两个相差8字节

## 中断请求的撤销
1. TIM中断撤销
    - TIM0/1中断自动撤销
    - TIM2中断需要手动撤销 -> CLR TF2+CLR EXF2
2. INFO中断在两种触发方式下都自动撤销
3. USART中断撤销
    - CLR TI,RI

## PPT
1. 中断地址要记住
    - 程序存储ROM，不能占用中断位置
2. I<>,T<>,E<>,P<>
    - I<>:中断控制寄存器 -> INFO
    - T<>:中断控制寄存器 -> TIM
    - E<>:中断允许寄存器
    - P<>:中断优先级寄存器
3. 中断级别顺序（同级别时）
    - INF0>TIM0>INF1>TIM1>USAT>TIM23
4. 周期
    - 外部中断最短响应时间：3个机械周期，最长8个机械周期
    - 最短：中断查询标志位占，1个机械周期，该机械周期处于指令最后一个机械周期，接着调用LCALL，占两个机械周期
    - 最长：中断查询标志位占，1个机械周期，访问RET，1个机械周期，计算（乘除法）4个机械周期，调用LCALL，2个机械周期

6. 中断程序设计
```ASM
ORG 0000H
    LCALL MAIN
ORG 0003H
    LCALL INT
ORG 000BH
    LCALL TIM0

INT:
    中断事件    
```
