## 课后题
1. 单片机名称：单片微型计算机，（微控制器MCU）,（嵌入式微控制器EMCU）
2. 单片机通过（总线）将（CPU）（储存器）（I/O口）集成在一块芯片上
3. AT89C52的工作频率上限是（33）MHz
4. 专业单片机降低（成本，提高可靠性）
5. 单片机采用二进制的原因：<u>受器件物理性能的限制</u>
6. STC系列单片机是8051内核的单片机
7. C52比C51多了4KB的Flash，128B的RAM，1个中断源，1个定时器
