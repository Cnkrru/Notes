## 指令
1. 按字节分类：
    - 单字节
    - 双字节
    - 三字节
2. 按机械周期分类
    - 1个机械周期
    - 2个机械周期
    - 4个机械周期
3. 操作码与操作数
    - 操作码：命令
    - 操作数：可以是直接的数字，也可以是地址

## 寻址方式（7种）
1. 寄存器寻址
    - 操作指定寄存器存储的数值（寄存器的数值）
    - 寻址范围
        - 通用寄存器（R0-R7）
        - 特殊寄存器
2. 直接寻址
    - 操作数值对应的地址的数值（地址的数值）
    - 寻址范围
        - 片内RAM地址
        - 特殊寄存器
3. 寄存器间接寻址
    - 操作指定寄存器的数值所对应的地址的数值（寄存器内的地址的数值）
    - 寻址地址
        - 片内RAM地址
        - 片外RAM地址
        - 堆栈操作
4. 立即数寻址
    - 直接传递数
5. 基址寄存器+变址寄存器间接寻址（几乎不用）
    - 基址寄存器：DPTR,PC
    - 变址寄存器：A
    - 先把两个寄存器的地址相加，再按照地址寻址
    - 指令（3条）：
        - MOVC A,@A+DPTR
        - MOVC A,@A+PC
        - JMP @A+DPTR
6. 相对地址寻址
    - 操作相对地址的数值（-128~+127）
7. 位寻址
    - 操作指定地址的数值

>总结下来只有两类：地址寻址，寄存器寻址

## 指令集备注
1. MOV指令目标
    - A
    - Rn
    - direct
    - @Ri
    - DPTR/PC
2. MOVC
    - DPTR指针模板
        MOV A,#01H              //01H为下标索引
        MOV DPTR,#ARRAY         //将数组入口地址传给数据指针
        MOVC @A,A+DPTR          //根据索引拿到地址
    - PC指针设计思路
        >PC指针查表依赖程序位置，需要从当前位置计算出与数组地址之间的距离，补全距离来查表
        >一般情况下，查表用DPTR指针

## 指令集
1. 赋值指令
    - MOV <地址/寄存器>,<地址/寄存器>
        >MOV用于访问片内RAM
    - MOVX <地址/寄存器>,<地址/寄存器>
        >MOVX用于访问片外RAM
        >要配合RD,WR引脚使用
    - MOVC <地址/寄存器>,<地址/寄存器>
        >MOVC用于访问ROM中连续地址预设的数组（查表）
        >唯一可访问ROM的数据指令
        >只读类型
        >一般采用相对地址寻址，只有两条指令：
            - MOVC A,@A+PC      -> 在A+PC+1处运行
            - MOVC A,@A+DPTR    -> 在A+DPTR处运行
        >要伴随PSEN引脚使用
2. 堆栈指令
    - PUSH <寄存器>
        >PUSH用于压栈
    - POP <寄存器>
        >POP用于出栈
3. 字节交换指令
    - XCH <地址/寄存器>,<地址/寄存器>
        >XCH用于交换两个寄存器/地址的数值
    - XCHD <地址/寄存器>,<地址/寄存器>
        >XCHD用于交换两个寄存器/地址的数值的低4位
4. 加增指令
    - ADD <地址/寄存器>,<地址/寄存器>
        >ADD用于加法运算
    - ADDC <地址/寄存器>,<地址/寄存器>
        >ADDC用于带进位加法运算
    - INC <地址/寄存器>
        >INC用于自增运算
    - DA <地址/寄存器>
        >DA用于十进制自动校准
4. 减减指令
    - SUB <地址/寄存器>,<地址/寄存器>
        >SUB用于减法运算
    - SUBB <地址/寄存器>,<地址/寄存器>
        >SUBB用于带借位减法运算
    - DEC <地址/寄存器>
        >DEC用于自减运算
    - DA <地址/寄存器>
        >DA用于十进制自动校准
5. 乘除指令
    - MUL <地址/寄存器>,<地址/寄存器>
        >MUL用于乘法运算
    - DIV <地址/寄存器>,<地址/寄存器>
        >DIV用于除法运算
6. 累加器指令(含有部分通用指令)
    - CLR A
        >clear A
    - CPL A
        >将A中0/1取反
    - RL A
        >将A中数字左环移1位
    - RLC A
        >将A与Cy中的数字左环移
    - RR A
        >右环移
    - RRC A
        >带进位右环移
    - SWAP A
        >A半字节交换指令
7. 逻辑指令
    - ANL <寄存器/地址>,<急存器/地址>
        >与
    - ORL <寄存器/地址>,<寄存器/地址>
        >或
    - XRL <寄存器/地址>,<寄存器/地址>
        >异或
8. 跳转指令
    - LJMP <地址>
        >全局跳转
    - LJMP <相对地址>
        >相对跳转
    - AJMP <>

10. NULL
    - NOP
        >空指令，执行PC+1
11. 位指令
    - MOV
    - CLR
    - CPL
    - SETB <寄存器/地址>
        >指定位置置1
    - ANL,ORL
    - JC,JNC,JB,JNB,JBC 
12. 伪指令
    - ORG <地址>
        >程序起始地址
    - END 
        >程序结束标志
    - EQU <变量> <值>
        >给变量赋值

        2. DW <值><值><值><值><值>

            >从指定起始地址保留指定数值的地址数
