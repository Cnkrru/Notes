## 定时器结构
1. 3个定时器:T0,T1,T2
    - T0:由TH0,TL0构成
    - T1:由TH1,TL1构成
2. 计数器每次计数从初始值开始计数
    - 复位后，初值为0
## 工作模式
1. TMOD寄存器
    - D0-D3:T0 -> M0,M1,C/T,GATE
    - D4-D7:T1 -> M0,M1,C/T,GATE
2. GATE
    - GATE=0,只由运行控制位TRx控制定时器/计时器运行
    - GATE=1,用外部中断引脚的电平与TRx共同控制
3. M0,M1
    - 00:13位定时器/计时器
    - 01:16位定时器/计时器
    - 10:8位自动重装定时器/计时器
    - 11:仅适用于T0,T0分成两个8位计数器，T1停止计数
4. C/T
    - 0:T0,T1为定时器
    - 1:T0,T1为计数器

## TCON控制寄存器
1. TF0/1
    - TCON.5 -> TF5
    - TCON.7 -> TF1
    - 计数器溢出时，该位置1
2. TR0/1
    - TCON.4 -> TR0
    - TCON.6 -> TR1
    - 0:T0,T1停止计数
    - 1:T0,T1开始计数
## 工作方式
1. 00
    - L5+H8
    - L5溢出向H8进位，H8溢出向TF进位
    - 参数
        - 计数范围:1~2^13
        - 定时范围:1个机械周期~2^13个机械周期
        - 计数个数:2^13-初值
        - 定时时间:（2^13-初值）*机械周期
2. 01
    - L8+H8
    - 同00方式
    - 参数
        - 计数范围:1~2^16
        - 定时范围:1个机械周期~2^16个机械周期
        - 计数个数:2^16-初值
        - 定时时间:（2^16-初值）*机械周期
3. 10
    - L8（H8作为初值预设器）
    - 当L8溢出时，TF置1，H8将预设值给L8
    - 参数
        - 计数范围:1~2^8
        - 定时范围:1个机械周期~2^8个机械周期
        - 计数个数:2^8-初值
        - 定时时间:（2^8-初值）*机械周期
4. 11
    - T0工作，T1不工作
    - T0拆分为TH0,TL0
    - 一般用于波特率发生器
## 时间计算
- 震荡频率：f
- 震荡周期：1/f
- 机械周期：12/f
- 计数个数：max-初值
- 定时时间：计数个数*T机械
- 模式最大定时时间：计数max*T机械
    - 00：16.384ms
    - 01：131.072ms
    - 10：512us

## PPT例题
1. 已知工作方式，寄存器情况求计数个数/定时周期
    - 计数个数=max-初值
    - 定时周期=计数个数*T机械
2. 已知工作方式，定时周期，求初值
    - 计数个数=定时周期/T机械
    - 初值=max-计数个数
3. 程序设计
    - 已知工作方式，频率
    - 代码模板    
    ```asm
    ORG 0000H
    SJMP MAIN

    ORG 001BH
    SJMP TIM1

    ORG 0040H

    MAIN:
        MOV TMOD,#20H       ;20H -> 0010 0000 表示寄存器内部状态
        MOV TH1,#56H        ;56H -> 200us，初值为56，256-56
        MOV TL1,#56H        ;56H -> 200us，同上

        SETB EA             ;允许中断
        SETB ET1            ;允许溢出，溢出则触发中断子程序
        SETB TR1            ;启动定时器
        SJMP $              ;while（1） $相当于本条指令的地址，无限循环

    TIM1:
        CPL P1.0
        RETI
    ```
>例题1：每计数100，反转电平

>例题2：10ms延时，流水灯
```asm
ORG 0000H
SJMP MAIN

ORG 001BH
SJMP TIM1

ORG 0040H

;初始化函数
MAIN:
    MOV A,#0FEH         ;0000 1111 1110
    MOV P1,A

    MOV TMOD,#10H       ;0001 0000
    MOV TH1;0D8H        ;高8位
    MOV TL1,0F0H        ;低8位

    SETB EA
    SETB ET1
    SETB TR1
    
    SJMP $
TIM1:
    RL A
    MOV P1,A
    MOV TH1,0DBH
    MOV TL1,0F0H
    RETI
    END
```

>例题3：系统6MHz，编写1s延时
```asm
ORG 0000H
SJMP MAIN

ORG 001BH
SJMP TIM1

ORG 0030H

MAIN:
    MOV R7,#10

    MOV TMOD,#01H
    MOV TH1，#3CH
    MOV TL1，#0B0H

    SETB EA
    SETB ET1
    SETB TR1
    
    SJMP $
TIM1:
    MOV TH1，#3CH
    MOV TL1，#0B0H
    DJNZ R7,RETURN
    CPL P1.0
    MOV R7,#10
RETURN:
    RETI
    END
```

>例题4：产生3：1（高：低）
```asm
ORG 0000H
SJMP MAIN
ORG 001BH
SJMP TIM1
ORG 0030H

MAIN:
    MOV R7;#3           ;高电平循环次数
    MOV TMOD,#02H
    MOV TH1,#206        ;100US
    MOV TL1,#206        ;100US

    SETB    EA
    SETB    ET1
    SETB    TR1
    SJMP    $
TIM1:
    JB      P1.0 LOW            ;如果为高位，跳转HIGH
    SETB    P1.0                ;否则，设置P1.0为高位
    SJMP    RETURN              ;跳转到RETURN
LOW:
    DJNZ    R7,RETURN           ;循环3次,不为0跳出该程序，去执行高电平程序
    CLR     P1.0                ;低电位
    MOV     R7,#3               ;R7=0，重置高电平循环次数
RETURN:
    RETI
    END
```
