## 方波

;方波：给A赋值，传给DPTR，然后DELAY
```asm
ORG 0000H
MAIN:
    MOV DPTR,#0000H
LOOP:
    MOV A,#0FFH
    MOVX @DPTR,A 
    LCALL DELAY
    MOV A.#00H 
    MOVX @DPTR,A 
    LCALL DELAY
    SJMP LOOP
DELAY:
```
