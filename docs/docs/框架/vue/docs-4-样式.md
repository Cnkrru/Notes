## 样式

### 绑定方式
>:class=>绑定选择器

>:style=>绑定CSS属性
- 这两种方式对应原生HTML+CSS的`class`和`style`属性，只是在此基础上进行增强

---
### 增强效果
- class(传递响应式数据，动态切换选择器)
    1. 可以接收基础类型和对象类型
    2. 可以接收计算属性
    3. 可以接收数组(对应HTML绑定多个选择器)
- style(传递响应式数据，动态切换CSS属性)
    1. 可以直接在HTML中使用
    2. 可以在`<script setup>`中定义reactive对象
    3. 可以接收计算属性
    4. 可以接收数组
---
### 个人看法
> 其实和原本的CSS写法差不多，只是增强了响应式数据的功能，把原本需要配合js实现的功能简化了
